# 🧩 AI Puzzle Solver

An AI agent that autonomously solves puzzle problems
using search algorithms and heuristic techniques.

## 🛠️ Technologies Used
- Python
- Search Algorithms (BFS / DFS / A*)
- Google Colab

## ⚙️ What I Did
- Implemented an AI agent to solve puzzle problems
- Applied uninformed and informed search algorithms
- Optimized solution path using heuristic evaluation
- Measured performance by number of steps and time taken
